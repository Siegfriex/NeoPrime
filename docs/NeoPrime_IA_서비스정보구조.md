## 1. 서비스 개요

### 1.1 플랫폼 구성

| 플랫폼 | 대상 사용자 | 주요 목적 |
|:------:|:-----------|:----------|
| **웹 대시보드** | 원장, 강사 | 학원 운영, 학생 관리, 평가 입력, 분석 |
| **모바일 앱** | 학생, 학부모 | 성과 확인, 합격 진단, 피드백 조회 |

### 1.2 사용자 페르소나

| 페르소나 | 역할 | 핵심 니즈 |
|:---------|:-----|:----------|
| **원장** | Elite Academy Partner | 라인 잡기, 합격 예측, 설명회 자료, 강사 관리 |
| **강사** | 실기 파트 리더/수석 강사 | 주간 평가 입력, 학생 성장 추적, AI 피드백 활용 |
| **학생** | B2B2C 최종 사용자 | 주간 성과 확인, 합격 가능성, 액션 플랜 |
| **학부모** | B2B2C 보호자 | 자녀 현황, 합격 진단, 상담 준비 |

### 2.2 정보 계층 구조

| Level | 웹 대시보드 | 모바일 앱 |
|:-----:|:-----------|:----------|
| **L0** | NeoPrime 플랫폼 | NeoPrime 앱 |
| **L1** | 대시보드, 학생관리, 평가관리, 분석, 설정 | 홈, 성과, 합격진단, 성공사례, 내정보 |
| **L2** | 리스트/상세/입력 페이지 | 리포트/그래프/상세 페이지 |
| **L3** | 필터/검색/모달 | 필터/상세모달 |

## 3. 사이트맵

### 3.1 웹 대시보드 사이트맵

```
/                                    → 대시보드 홈
│
├── /students                        → 학생 관리
│   ├── /students/list              → 전체 학생 리스트
│   ├── /students/:id               → 학생 상세 프로필
│   ├── /students/new               → 학생 추가
│   └── /students/:id/edit          → 학생 수정
│
├── /evaluations                     → 평가 관리
│   ├── /evaluations/new            → 주간 평가 입력
│   ├── /evaluations/history        → 평가 이력 조회
│   └── /evaluations/ai-feedback    → AI 피드백 생성기
│
├── /analytics                       → 분석 & 리포트
│   ├── /analytics/admission        → 합격 예측 분석
│   ├── /analytics/growth           → 성장 분석
│   ├── /analytics/teacher-bias     → 강사 편차 분석
│   └── /analytics/reports          → 리포트 다운로드
│
├── /settings                        → 설정
│   ├── /settings/account           → 계정 설정
│   ├── /settings/academy           → 학원 정보
│   ├── /settings/teachers          → 강사 관리
│   └── /settings/data              → 데이터 관리
│
└── /auth                            → 인증
    ├── /auth/login                 → 로그인
    ├── /auth/logout                → 로그아웃
    └── /auth/forgot-password       → 비밀번호 찾기
```

### 3.2 모바일 앱 사이트맵

```
/                                    → 홈 (주간 성과 요약)
│
├── /performance                     → 성과
│   ├── /performance/weekly         → 주간 리포트
│   ├── /performance/growth         → 성장 그래프
│   └── /performance/feedback-history → 피드백 이력
│
├── /admission                       → 합격 진단
│   ├── /admission/targets          → 목표 대학 선택
│   ├── /admission/probability      → 합격 확률 확인
│   └── /admission/similar-cases    → 유사 사례
│
├── /success-stories                 → 성공 사례
│   ├── /success-stories/:id        → 사례 상세
│   └── /success-stories/ask        → 질문하기
│
├── /profile                         → 내 정보
│   ├── /profile/edit               → 프로필 수정
│   ├── /profile/targets            → 목표 대학 설정
│   └── /profile/notifications      → 알림 설정
│
└── /auth                            → 인증
    ├── /auth/login                 → 로그인
    └── /auth/logout                → 로그아웃
```
