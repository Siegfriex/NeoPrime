# pandas 및 필수 패키지 설치 확인 가이드

## 빠른 확인 방법

다음 명령어로 pandas 설치 여부를 확인할 수 있습니다:

```bash
# 방법 1: 간단한 확인
python -c "import pandas; print(f'pandas 버전: {pandas.__version__}')"

# 방법 2: 체크 스크립트 사용 (권장)
python check_pandas.py
```

## 설치 상태 확인

### 현재 Python 환경
- **Python 경로**: `Y:\0126\python_env.env`
- **설정 파일**: `run_system.bat`에서 자동으로 경로 설정됨

### 필수 패키지 목록
`requirements.txt`에 정의된 패키지:
- `pandas>=2.0.0` - 데이터 처리
- `openpyxl>=3.1.0` - 엑셀 파일 읽기 (.xlsx)
- `pyarrow>=14.0.0` - Parquet 형식 지원
- `google-cloud-bigquery>=3.11.0` - BigQuery 클라이언트
- `google-auth>=2.23.0` - 인증
- `google-auth-oauthlib>=1.1.0` - OAuth 인증

## 설치 방법

### 1. 전체 패키지 설치 (권장)
```bash
cd Y:\0126\0126
pip install -r requirements.txt
```

### 2. 개별 패키지 설치
```bash
pip install pandas openpyxl pyarrow google-cloud-bigquery google-auth google-auth-oauthlib
```

### 3. 설치 확인
```bash
python check_pandas.py
```

## 문제 해결

### pandas가 설치되지 않은 경우
```bash
# pip 업그레이드 후 재설치
python -m pip install --upgrade pip
pip install pandas
```

### 권한 오류 발생 시
```bash
# 사용자 디렉토리에 설치
pip install --user -r requirements.txt
```

### 특정 패키지 설치 실패 시
```bash
# 개별 설치로 문제 패키지 확인
pip install pandas
pip install openpyxl
pip install pyarrow
```

## 확인 명령어

python -c "import pandas; print(pandas.__version__)"
