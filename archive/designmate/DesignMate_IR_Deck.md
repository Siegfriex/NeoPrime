## 슬라이드 10: Problem - 페인포인트 1

### Pain Point 1

#### 타이틀
**실력 진단 불가**

#### 문제 설명
- **1차 문제**: "내 작품이 어느 수준인가?" 주관적 판단만 가능
- **2차 문제**: 실기 선생님의 평가가 전부 → 다른 시각의 평가 없음
- **3차 영향**: 정량적 기준 부재 → 성장 추적 어려움, 자신감 부족

#### 핵심 메시지
"몇 점짜리 실력인가?" 명확한 답이 없음

## 슬라이드 21: Product - Key Features

### Key Features

**핵심 기능 4가지**

#### 기능 1: Vision AI 작품 분석
- **타이틀**: 8초 만에 5개 지표 정량화
- **상세**:
  - Gemini 3 Pro Vision 사용
  - 밀도/형태력/완성도/정합성/사고력
  - 0-100점 + 등급 (S/A+/A/A-/B/B-/C)
  - Bauhaus/Swiss 디자인 이론 기반
- **현황**: ✅ dysapp 기술 이식

#### 기능 2: 입시 DB 대학 라인 추정
- **타이틀**: 226,695행 입시 데이터 활용
- **상세**:
  - NeoPrime Theory Engine 패턴 재사용
  - 성적 + 작품 메트릭 → 합격 확률
  - TOP/HIGH/MID/LOW 라인별 대학 추천
  - 건대/동국대/경희대 등 40+개 대학
- **현황**: 🚧 개발 예정 (M3)

#### 기능 3: fixScope AI 멘토
- **타이틀**: 우선순위화된 피드백
- **상세**:
  - StructureRebuild: 구조 재설계 집중
  - DetailTuning: 마감/디테일 집중
  - 액션 플랜 3-5개 자동 생성
  - Gemini 2.5 Flash 챗봇
- **현황**: 🚧 개발 예정 (M4)

#### 기능 4: 성장 추적
- **타이틀**: 주차별 실력 향상 시각화
- **상세**:
  - 5개 지표별 성장 곡선
  - 작품 비교 (현재 vs 이전)
  - 라인 변화 추적 (MID → HIGH)
  - 벡터 검색 (유사 합격작 추천)
- **현황**: 🚧 개발 예정 (M5)

  - 61-80: HIGH 라인 (건대/동국대)

**형태력 (Shape Power, 0-100)**
- 정의: 기본 도형을 정확하게 그리는 능력
- 측정: 원근, 비례, 구조 안정성, 선 명확성
- 입시 기준:
  - 61-80: HIGH 라인
  - 81-100: TOP 라인

**완성도 (Completion, 0-100)**
- 정의: 마감/디테일/밸런스의 완성 정도
- 측정: 마감 정도, 디테일, 전체/부분 밸런스, 톤 통일
- 입시 기준:
  - 61-80: HIGH 라인
  - 81-100: TOP 라인

**정합성 (Coherence, 0-100)**
- 정의: 주어진 문제를 얼마나 잘 이해하고 표현했는가
- 측정: 주제 표현, 컨셉 명확성, 창의성
- 입시 기준:
  - 61-80: HIGH 라인
  - 81-100: TOP 라인

**사고력 (Thinking Power, 0-100)**
- 정의: 시각적 스토리텔링/문제해결능력
- 측정: 시선 흐름, 강조 포인트, 계층 구조, 전달력
- 입시 기준:
  - 61-80: HIGH 라인
