# NeoPrim: 실기 평가 기반 예체능 입시 AI 엔진
## 최종 Product Requirements Document (PRD)

---

## Executive Summary

**제품명**: NeoPrim (네오프림)  
**서브라인**: "원장의 눈을 AI가 배운다"  
**핵심 가치**: 실기 평가 기반 예체능 입시 전문 B2B SaaS  
**1차 MVP 타겟**: 미대입시 전문 학원 (강남 및 전국 상위권 학원)  
**2차 확장 타겟**: 체대 / 음대 / 연영 입시 학원 (예체능 전역)  
**시장 기회**: 국내 에듀테크 시장 11조원(2026 예상) 중 **예체능 입시 세그먼트 1,200억~1,500억원** (미개척)  
**BM 1차**: 학원 매출 증대 도구 (SaaS 라이선스, 월 300~500만원)  
**BM 2차**: 예체능 입시 AI 플랫폼화 (4개 모듈, 월 10억원 규모 목표)

## 1. 시장 & 환경 분석

### 1.1 매크로 시장 현황

#### 1.1.1 국내 교육시장 규모 및 성장

| 지표 | 2022 | 2025 | 2026 (예상) | CAGR |
|-----|------|------|-----------|------|
| **전체 에듀테크 시장** | - | 9.9조원 | 11.0조원 | 8.5% |
| **사교육비 총액** | 26.0조원 | - | - | - |
| **사교육 참여율** | 78.3% | - | 80% 이상 | - |

**핵심 인사이트**:
- 2022년 사교육비 26조원 중 **일반 교과과정: 31만원/월** vs **예체능: 14만원/월**[web:189]
- 예체능 사교육비도 **YoY 13.0% 증가 (가장 빠른 성장)**[web:189]
- 에듀테크는 초기 단계지만, 공교육 보다 사교육 시장 위주로 발전[web:189]

#### 1.1.2 예체능 입시 시장의 특수성

**예체능 계열은 "정시로만 불가능"한 시장**:
- 서울대/홍대/이대 등 상위권 대학: **실기 + 수능 + 내신/포트폴리오 통합 평가**[web:190]
- 하위권 대학: **실기 비중 70~100%** (거의 순수 실기 선발)[web:196]
- AI 입시 서비스(서울런, AIM, 바이브온 등) **모두 수능/정시/학종 중심** → 실기 기반 예체능 무시[web:165][web:168][web:174][web:191]

**경쟁률 추이 (2025~2026)**:
- 미대: 서울대 미대 회화 5.5대 1, 조소 6.16대 1[web:193]
- 음대: 서울예대 실용음악 68.8대 1 (악기별 1~2명만 선발)[web:193]
- 체대: 상위권은 수능 높음, 하위권은 실기 중심[web:190]

→ **"정시(수능)로는 안 되고, 실기가 절대적"이라는 특수 시장**

#### 1.1.3 미대 입시의 최신 트렌드 (2025~2026)

**1) 비실기전형 확대 (수시)**
- 서울대, 홍대, 고려대, 이화여대 등 상위권이 비실기 종합전형 도입[web:190]
- 1,200명 이상을 비실기(서류+면접)로 선발[web:190]
- 하지만 실기전형은 여전히 **서울대/홍대 등 상위권의 핵심 채용 통로**

**2) 수능 비중 증가 (정시)**
- 미대 정시도 수능 비중 80% 이상 대학 증가[web:190]
- 예외: 홍대, 국민대 등은 여전히 실기 50~60% 반영

**3) 기초소양/포트폴리오 중심화**
- "그림 실력" 자체보다 "사고력/창의력/표현력" 평가 강화[web:190]
- 겨울특강(실기 집중), 주간 평가가 **"입시 성공의 중요한 지표"**가 됨

#### 1.1.4 예체능 입시 학원 시장 규모 (데스크리서치)

| 카테고리 | 학원 수 (추정) | 평균 월 매출 | 세그먼트 시장 |
|---------|----------|-----------|-----------|
| **미술 입시 학원** | 500~700곳 | 5,000~20,000만원 | 300~1,400억원 |
| **체대 입시 학원** | 300~400곳 | 3,000~10,000만원 | 90~480억원 |
| **음악 입시 학원/레슨센터** | 1,000~1,500곳 | 2,000~8,000만원 | 200~1,200억원 |
| **연기/예영 입시 학원** | 200~300곳 | 3,000~10,000만원 | 60~360억원 |
| | | | **총: 650~3,440억원** |

**설정 근거**:
- 수도권 및 광역시 중심 상위권 학원만 고려 (AI 도입 가능성 높음)
- 강남권 미술학원 300곳 기준: 월 평균 1,000만원 이상 + 실기 평가 시스템 보유
- 체대/음대 학원은 더 많지만 분산도 높음

### 1.2 경쟁 환경

#### 1.2.1 직접 경쟁 (AI 기반 예체능 입시 서비스)

**현황**: 거의 없음

| 서비스명 | 타겟 | 특징 | 한계 |
|---------|------|------|-----|
| **서울런 AI** | 일반 + 예체능 | 1,220만 건 합격 데이터 | **실기 평가 무시**, 정시/수능 중심 |
| **AIM** | 수능 위주 | 수십만 건 데이터 | **예체능 실기 배제** |
| **바이브온** | 학종/수시 | 1,240만 문장 생활기록부 + 12만 건 합격 | **비실기 학종 중심**, 실기 평가 없음 |
| **큐피드 AI (이투스)** | 학종 + 컨설팅 | 생활기록부 분석 + 탐구보고서 | **일반 대학생 위주**, 미대/체대 특화 없음 |

→ **예체능 + 실기 기반 평가 엔진 = 완전 미개척 시장**[web:165][web:168][web:171][web:174][web:191][web:194]

#### 1.2.2 간접 경쟁 (전통 입시 컨설팅)

- **대면 컨설턴트**: 시간당 50만원, 대도시 집중, 주관성 높음[web:191]
- **학원 자체 시스템**: 엑셀/종이 기반, 전국 확장 불가, 신규 강사 기준 편차
- **일반 에듀테크**: 미대/체대 특화 없음, 실기 평가 불가

**NeoPrim의 차별점**: 
1. **유일한 "실기 평가 기반" AI 모델**
2. **예체능 전문가(원장) 암묵지 학습 + AI 객관화**
3. **학원 매출 증대로 ROI 입증 가능**
4. **전국 확장성 높음** (데이터 누적 구조)

### 1.3 SWOT 분석 (NeoPrim)

#### STRENGTH (강점)

1. **네오캣 파트너십의 검증된 데이터**
   - 서울대/홍대/이대 중심 **200명+/년 × 다년간 합격 데이터**
   - 원장의 주간 평가(A~F) + 최종 결과 완벽한 연결 가능
   - 한국 미대입시에서 **가장 신뢰도 높은 실제 데이터**

2. **독점적 기술: 원장 암묵지 학습**
   - "원장이 그림을 보고 A~F를 주는 기준"을 AI가 모사 가능
   - 경쟁사가 절대 따라올 수 없는 도메인 특화 모델

3. **확장 가능한 패턴**
   - 미대 → 체대/음대/연영으로 "실기 평가 기반" 패턴 그대로 재사용
   - 각 도메인은 평가 항목만 다름 (구도→체력, 톤→음정, 등)

4. **시장 진입의 현실성**
   - 미대 입시는 **상위권 집중도 높음** (강남, 서울, 수도권)
   - 네오캣 같은 **강남 기반 명문학원 = 좋은 레퍼런스**

#### WEAKNESS (약점)

1. **초기 데이터 스케일 제약**
   - 네오캣 1곳만으로는 200명/년 → 전국 추론 어려움
   - 2~3년 누적 필요 (모델 안정성 확보)

2. **개인정보 민감성**
   - 학생별 실기 평가/합격 결과 = 민감 정보
   - 강화된 암호화 + 동의 프로세스 필수

3. **도메인 전문성 의존**
   - 각 영역(미대/체대/음대) 원장의 협력 필수
   - 1차 MVP 후 빠른 파트너 영입 필요

#### OPPORTUNITY (기회)

1. **예체능 입시 AI = 완전 미개척 시장**
   - 에듀테크 11조원 중 예체능 실기 기반 AI는 1%도 미개척
   - 정부의 "AI 교육 활성화" 정책 + 교육청 공공 플랫폼 기회

2. **고단가 B2B SaaS 가능성**
   - 학원 입장: 합격률 5% ↑ = 월 1~2억 추가 매출 가능
   - 월 400~500만원 SaaS = 3~4개월 내 ROI 회수 (매우 매력적)

3. **인프라 & AI 기술 성숙**
   - Vertex AI Tabular, Vision, LLM 모두 한국에서 쉽게 접근 가능
   - 기술 장벽 낮음, 도메인 지식이 핵심

4. **2차 B2B2C 확장 기회**
   - 체대/음대/연영 모두 동일 "실기 평가 + 결과" 구조
   - 각 영역 학원 × 50~200곳 = 월 10억원 규모 가능

#### THREATS (위협)

1. **대형 에듀테크의 예체능 진입**
   - 서울런, 바이브온이 실기 모듈 추가할 수 있음
   - 하지만 "원장 암묵지" 부재로 깊이 한계 (2~3년 선점 기회)

2. **데이터 규제 강화**
   - 개인정보보호법 강화 → 학생 데이터 사용 기준 변화
   - 초기부터 강화된 privacy-by-design 필수

3. **학원 경영 악화**
   - 학령인구 감소 → 일부 하위 학원 폐업
   - 상위권 학원만 지속 가능 (NeoPrim은 상위권 중심이므로 상대적으로 안전)

## 2. 현재 보유 자산 분석

### 2.1 기존 엑셀 기반 고속성장 분석 모델

**구성**:
- **Index 시트**: 수능 영역별 백분위 → 표준점수 환산 (100만+ 행)[file:119]
- **Subject 시트**: 과목별 난이도/분포 (연도별)[file:119]
- **Percentage 시트**: 수험자 분포 데이터 (누적 백분율)[file:119]
- **수계산 엔진**: 학생 입력(수능 성적 → 백분위 → 합격선 추정)[file:119]

**현재 용도**: 수능/정시 기반 합격 예측 (일반 대학)

**NeoPrim에서의 재사용**:
- 정시 세그먼트 예측 (실기 + 수능 결합)
- 학생의 "정시 라인" + "수시 실기 라인" 통합 조언
- 기초 데이터 정규화 (수능 난이도 보정)

### 2.2 네오캣과의 MOU를 통해 얻을 수 있는 데이터 자산

#### 2.2.1 학생 기본 데이터

```
student_id, name, school, grade_level, major_category
예시:
S001, 김○○, 강남고, 3, 디자인
S002, 이○○, 서울고, 3, 미술이론
```

**스케일**: 현재/과거 200명/년 × 다년도 = **1,000~2,000명 누적**

#### 2.2.2 원장 주간 평가 데이터 (CRITICAL)

```
eval_id, student_id, week_number, eval_date, course_type, subject_matter
eval_score, score_a (구도), score_b (톤), score_c (명암), score_d (발상), score_e (완성도)
예시:
E0001, S001, 1, 2025-11-24, 겨울특강, 수묵화, A-, 9, 8, 8, 9, 8.5
E0002, S001, 2, 2025-12-01, 겨울특강, 정물소묘, B+, 8, 7, 7, 8, 7.5
...
E0008, S001, 8, 2026-01-05, 겨울특강, 인물드로잉, A, 9.5, 9, 8.5, 9.5, 9
```

**스케일**: 200명 × 8주 = **1,600개 평가 레코드/년**

**가치**:
- "원장이 그림을 보고 실시간 주는 피드백"의 완전한 기록
- 학생별 성장곡선(B- → B → B+ → A-) 추적 가능
- 각 평가 항목별 상관성 분석 가능

#### 2.2.3 최종 합격/예비/불합격 데이터

```
result_id, student_id, season (수시/정시), university, major
result_status (합격/예비/불합), registered_yn (등록여부)
예시:
R001, S001, 수시, 홍대, 디자인, 합격, Y
R002, S002, 수시, 서울대, 회화, 예비, N
R003, S003, 정시, 이대, 조소, 불합격, N
```

**스케일**: 200명 × 3지망 = **~600개 결과 레코드/년**

**가치**:
- "원장 평가 A- ⟹ 실제 홍대/이대 합격" 확률 정량화 가능
- "B+ 수준 → 지원 전략" 피드백 가능
- 대학/학과별 "실제 입시 컷"을 데이터로 증명

#### 2.2.4 강사 평가/피드백 메모

```
feedback_id, student_id, eval_id, teacher_name, feedback_text, recommendation
예시:
FB001, S001, E0001, 강○○, "구도 감각은 우수, 음영 처리에서 경직됨. 
       부드러운 선 연습 필요", "명암 스터디 집중 추천"
```

**스케일**: 평가당 1~2개 메모 = **1,600~3,200개 피드백/년**

**가치**:
- "양적 점수" + "정성 피드백" 결합 → LLM으로 개인화 조언 가능
- "이런 강점을 가진 학생은 어떤 대학이 맞나" 학습 데이터

### 2.3 데이터 통합 스키마 (NeoPrim용)

**4개 핵심 엔티티**:

```python
# 1. Students
class Student:
    student_id: str
    name_anonymized: str  # 개인정보 보호
    school_code: int
    grade_level: int
    major_category: str  # 디자인/회화/조소/일러스트 등
    join_date: date
    hopeful_universities: List[str]
    
# 2. Weekly Evaluations
class WeeklyEvaluation:
    eval_id: str
    student_id: str
    week_number: int  # 1-8 (겨울특강)
    eval_date: date
    course_type: str  # 겨울특강/정규/특강
    subject_matter: str  # 수묵화/소묘/정물/인물 등
    
    # 평가 점수 (A+~F 또는 4.5~1.0)
    overall_grade: str  # "A-", "B+", etc.
    
    # 세부 평가 항목 (선택)
    composition_score: float  # 구도 (0-10)
    tone_score: float  # 톤 (0-10)
    shading_score: float  # 명암 (0-10)
    concept_score: float  # 발상 (0-10)
    completion_score: float  # 완성도 (0-10)
    
    # 강사 피드백
    teacher_name: str
    feedback_text: str
    
# 3. Final Results
class AdmissionResult:
    result_id: str
    student_id: str
    season: str  # 수시/정시
    university: str
    major: str
    result_status: str  # 합격/예비/불합격
    registered: bool
    
# 4. Student Features (aggregate)
class StudentProfile:
    student_id: str
    
    # 기본 정보
    school_rank: str  # 일반고/자사고/특목고
    admission_academic_record: float  # 내신 or 모의고사 백분위
    
    # 수능 데이터 (정시 세그먼트용)
    csat_korean: int
    csat_math: int
    csat_english: int
    csat_total_percentile: float
    
    # 실기 누적 데이터
    eval_count: int
    avg_grade: float
    grade_trajectory: List[float]  # 주차별 점수 추이
    grade_momentum: float  # 성장각도
    
    # 역사 데이터
    years_studied: int

```

### 2.4 네오캣과의 MOU 기회요인

#### 2.4.1 상호 이득

| 당사 (NeoPrim) | 네오캣 |
|-------------|--------|
| **데이터 습득** (200명/년, 다년 누적) | **자체 AI 엔진 보유** ("네오캣 전용 입시 AI") |
| **모델 훈련** (2~3년 집중) | **합격률 상향** (브랜딩 개선) |
| **확장 자금 조달** (시드 라운드) | **월 300~500만 수익** (AI 라이선스) |
| **2차 파트너 영입** (레퍼런스) | **분원 확장** (입시 성적 증명) |

#### 2.4.2 네오캣 입장의 기대효과

1. **합격률 상승 (5~10%)**
   - AI 기반 객관적 진단 → 학부모/학생 신뢰도 ↑
   - 신규 강사/분원의 기준 표준화 → 품질 관리 ↑
   - 예상 결과: 75개 학원 추가 신입생 유입 × 월평균 500만원 = **월 3.75억원 추가 매출**

2. **브랜딩 강화**
   - "AI 미대 입시 엔진을 자체 개발한 유일한 학원"
   - 강남 미술학원 중 가장 기술적 선도 이미지
   - PR/입시 설명회에서 "AI 활용 합격 전략" 강점 부각

3. **데이터 기반 경영**
   - 학생별 성장 추적 + 대학별 합격 확률 → 과학적 상담
   - 강사 평가 표준화 → 급여/성과급 공정성 강화

4. **신규 매출 (라이선스)**
   - 월 300~500만 × 12개월 = 연 3,600~6,000만원 (추가 수익)
   - 기존 매출 기반으로 "거의 공짜" 데이터 기여

### 3.2 핵심 기능 (Feature Set)

#### **Phase 1: 원장 평가 에뮬레이션 엔진**

**입력**: 학생 프로필 + 과제 정보

```
입력 예시:
{
  "student_id": "S001",
  "week": 4,
  "course_type": "겨울특강",
  "subject_matter": "정물소묘",
  "prior_grades": [8.5, 7.5, 8.0],  # 지난 3주 평점
  "school_code": 1001,  # 강남고
  "major_category": "디자인"
}
```

**처리 (Vertex AI Tabular)**:

```
타깃: 원장이 줄 평가 점수 (A+~F 또는 4.5~1.0)
입력 피처:
  - 학생 기본정보 (학교, 계열, 입학월)
  - 과제 메타데이터 (유형, 난이도, 과제 기간)
  - 누적 성적 (이전 평가 점수, 성장률)
  - 시계열 (주차별 점수 추이)
  - (향후) 그림 이미지 임베딩 (비전 모델)

모델: Vertex AI Tabular Regression
  → 학습 데이터: 1,600 평가 × 다년도 = 5,000~10,000 샘플
```

**출력**: 예상 평가 점수 + 신뢰도

```
출력 예시:
{
  "predicted_grade": "B+",
  "confidence": 0.87,  # 87%
  "percentile_vs_cohort": 0.68,  # 같은 주차 학생 중 상위 68%
  "trend": "improving",  # 성장 추세
  "insight": "구도 감각은 평균 이상, 명암 처리 개선 필요"
}
```

#### **Phase 2: 합격 예측 모델**

**입력**: 학생 누적 데이터 + 희망 대학

```
입력 예시:
{
  "student_id": "S001",
  "final_week_grade": "A-",  # 8주차 최종 평가
  "grade_trajectory": [7.5, 8.0, 8.5, 8.5, 9.0, 9.0, 9.5, 9.0],
  "school_code": 1001,
  "admission_record": 3.5,  # 내신
  "csat_percentile": 0.75,  # 수능 예상 상위 25%
  "hopeful_universities": ["서울대", "홍대", "이대"]
}
```

**처리 (Multi-task Classification)**:

```
타깃 1: 합격 여부 (합격/예비/불합격, 각 대학별)
타깃 2: 대학 라인 추천 (Top/High/Mid/Low)
타깃 3: 합격 확률 (0~100%, 각 대학별)

입력 피처:
  - 최종 실기 수준 (A- = 상위 15%)
  - 실기 성장각도 (주차별 상승 추세)
  - 정시 라인 (수능 백분위)
  - 내신 (있을 경우)
  - 대학 특성 (입시 경쟁률, 실기 비중, 등급컷)
  - 학생 배경 (학교 수준, 선행학습 여부)

모델: Vertex AI Tabular Classification + Feature Importance
  → 학습 데이터: 600 결과 × 다년도 × 3대학/지망 = 1,800~3,600 샘플
```

**출력**: 합격 가능성 레포트

```
출력 예시:
{
  "student_name_anonymized": "S001",
  "admission_results": [
    {
      "university": "홍대",
      "major": "디자인",
      "probability": 0.82,  # 82% 합격률
      "recommendation": "HIGH",
      "reasoning": "최종 A- 수준, 홍대 입시 경쟁률 대비 충분한 실기 능력",
      "benchmark": "작년 A- 학생 중 85% 홍대 합격"
    },
    {
      "university": "서울대",
      "major": "회화",
      "probability": 0.45,
      "recommendation": "MEDIUM",
      "reasoning": "실기는 충분하나 면접/서류 경쟁 높음",
      "benchmark": "작년 A 수준 학생만 50% 합격"
    },
    {
      "university": "이대",
      "major": "미술",
      "probability": 0.92,
      "recommendation": "TOP",
      "reasoning": "실기 A- + 수능 우수 → 이대 입시 기준 충분",
      "benchmark": "작년 A- 학생 중 95% 이대 합격"
    }
  ]
}
```

#### **Phase 3: 성장 분석 & 시나리오 (Forecasting)**

**입력**: 지난 4주 성장 트렌드

```
입력 예시:
{
  "student_id": "S001",
  "recent_grades": [8.0, 8.5, 8.5, 9.0],  # 5~8주차
  "target_grade": "A",  # 목표: A 도달
  "weeks_remaining": 4  # 남은 주차
}
```

**처리 (Time-Series Forecasting)**:

```
모델: Prophet / Linear Regression
  → 현재 성장 속도 기반, 4주 후 예상 점수 계산
  → 목표 도달 가능성 판단
```

**출력**: 성장 시나리오

```
출력 예시:
{
  "current_trajectory": {
    "week_8_predicted": 9.2,  # 현재 추세면 9.2 도달 예상
    "confidence": 0.78,
    "message": "현재 성장속도면 1월 초 A 도달 가능"
  },
  "scenarios": [
    {
      "scenario": "BASELINE",
      "effort_level": "현재 수준 유지",
      "week_8_predicted": 9.2,
      "success_rate": 0.78
    },
    {
      "scenario": "INTENSIVE",
      "effort_level": "명암 집중 훈련",
      "week_8_predicted": 9.5,
      "success_rate": 0.92,
      "actions": ["명암 스터디 주 3회 → 5회 증대", "과제 양 50% 증가"]
    },
    {
      "scenario": "MINIMAL",
      "effort_level": "현재보다 약한 수준",
      "week_8_predicted": 8.8,
      "success_rate": 0.62
    }
  ]
}
```

## 9. 결론 & 비전

**NeoPrim**은:

1. **미개척 시장을 개척하는 세로우 AI**
   - 예체능 실기 평가 = 완전히 주관적이고, AI 해결책이 없음
   - "원장의 눈을 AI가 배운다"는 명확한 가치 제안

2. **강력한 초기 자산을 가진 스타트업**
   - 네오캣과의 파트너십 = 신뢰도 높은 데이터
   - 기존 고속성장 분석 엑셀 모델 = 기술 토대

3. **빠른 확장성이 높은 사업 모델**
   - 미대 → 체대/음대/연영으로 자연스러운 확장
   - 각 영역 100~200곳의 학원/센터 = 월 100억원 규모 가능

4. **교육 격차 해소에 기여하는 미션**
   - 대도시 집중된 입시 컨설팅을 AI로 전국화
   - 예체능 학생도 객관적이고 투명한 진단 가능
