- 아이콘: 돋보기+문서

- 서브: **Theory Engine으로 숫자로 전환**

## 슬라이드 24: Product - 6시간 개선 로드맵

### 즉시 실행 가능한 개선 계획

#### 6시간 로드맵

| 시간 | 작업 | 문제 해결 | 효과 |
|------|------|----------|------|
| **0-1h** | INDEX 우회 로직 구현 | INDEX 조회 실패 (0% 성공률) 해결 | +30% |
| **1-3h** | RAWSCORE 탐구과목 수정 | 탐구과목 점수 변환 실패 (40% 성공률) 해결 | +20% |
| **3-6h** | 대학명 Alias 추가 | 일부 대학 커트라인 미발견 (67% 성공률) 해결 | +5% |
| **6h** | **전체 통합 테스트** | **Golden Case 검증** | **95%+ 달성** |

#### 기술적 세부사항

**Issue #1: INDEX 우회 로직**
- 문제: INDEX 시트 인코딩 미해독
- 해결: RAWSCORE 누적% 합산으로 대체
- 소요: 1시간

**Issue #2: RAWSCORE 탐구과목**
- 문제: "영역" 컬럼에 국어/수학만 존재, 탐구과목 조회 실패
- 해결: "과목명" 컬럼 추가 매칭 로직

## 슬라이드 44: 부록 A - Theory Engine v3 기술 상세

### 부록 A: Theory Engine v3 기술 상세

#### 5단계 파이프라인 상세

**단계 1: RAWSCORE 변환**
- 입력: 원점수 (국어 80점, 수학 75점)
- 처리: RAWSCORE 시트 조회
- 출력: 표준점수 (125.0, 121.0), 백분위 (89.0, 82.0)

**단계 2: INDEX 조회**
- 입력: 표준점수 조합
- 처리: INDEX 시트 MultiIndex 빌드
- 출력: 누적백분위, 전국등수

**단계 3: PERCENTAGE 조회**
- 입력: 대학명, 전공, 계열, 누적백분위
- 처리: PERCENTAGE 시트 조회 + 보간
- 출력: 환산점수 (73.13, 610.87)

**단계 4: RESTRICT 체크**
- 입력: 학생 프로필 (영어 등급, 한국사 등급)
- 처리: RESTRICT 시트 결격 사유 검증
- 출력: 결격 여부 (True/False), 사유 리스트

**단계 5: 합격 라인 판정**
- 입력: 환산점수, 커트라인
- 처리: 확률 계산 모델
- 출력: 적정(80%+) / 예상(50-80%) / 소신(20-50%) / 상향(<20%)
